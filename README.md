# גיבוש תמ"צ באירוע חומ"ס | כבאות והצלה לישראל

**גרסה 2.0 - פשוטה ויציבה**

אפליקציה אינטראקטיבית לגיבוש תמונת מצב באירועי חומרים מסוכנים, מבוססת על התובנות והנחיות הרשמיות של כבאות והצלה לישראל.

## ✨ תכונות עיקריות

### 🔥 סימולציה אינטראקטיבית
- מעבר שלב אחר שלב דרך תהליך גיבוש התמ"צ
- אירוע חומ"ס מציאותי עם ממשק פשוט ואינטואיטיבי
- 3 שלבים: קבלת הודעה → איסוף מידע → בניית תמ"צ

### 🎯 שלושה צירים מקצועיים
1. **ציר שליטה** (אדום) - איסוף מידע מהמקורות השונים
2. **ציר מקצועי** (כחול) - מידע משלים על המתקן וסידורי הגנה מאש  
3. **ציר פיקודי** (ירוק) - מפקדים בזירה להשלמת המידע הדינמי

### 📊 מידע נדרש לתמ"צ ברורה
- מיקום וסוג האירוע
- זיהוי החומר/ים
- מאפייני הסיכון
- לכודים
- פעולות מפעליות

### 📄 מסמכים להורדה
- המצגת המלאה (PowerPoint)
- המסמך הרשמי (PDF)

## 🛠️ טכנולוגיות

- **Next.js 15** - React framework חדיש
- **React 19** - גרסה עדכנית
- **TypeScript** - Type safety
- **Tailwind CSS** - עיצוב מותאם
- **Lucide React** - אייקונים מקצועיים
- **ללא framer-motion** - פשוטה ויציבה

## 🚀 התקנה והרצה

```bash
# התקנת dependencies
npm install

# הרצה במצב פיתוח
npm run dev

# בניית production
npm run build

# הרצת production build
npm start
```
